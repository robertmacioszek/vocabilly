# Vocabilly

## Overview
Vocabilly is a lightweight Progressive Web Application designed to help users improve their vocabulary through interactive training sessions. This version is built using plain JavaScript and HTML, making it easy to understand and modify.

## Features
- Interactive vocabulary training sessions
- User input handling for tracking progress
- Local storage management for saving user sessions
- Offline support through service workers

## Project Structure
```
vocabilly
├── index.html         # Main HTML document
├── app.js             # JavaScript logic for the application
├── service-worker.js  # Service worker for PWA features
├── manifest.json      # PWA metadata
└── README.md          # Project documentation
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/robertmacioszek/vocabilly.git
   cd vocabilly
   ```

2. **Open the Application**
   Simply open the `index.html` file in your web browser to run the application.

## Usage
- Open the application in your browser.
- Start a training session by selecting a vocabulary list.
- Input your answers and track your progress.
